# PipeFX Design Mood Board

Visual direction based on the [Design Research Paper](file:///C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/design_research.md).

## 1. Overall Layout — The Bento Command Center

The primary interface uses a Bento grid with panels of varying sizes. The AI Chat is the largest panel (the main workspace). A narrow sidebar hosts the Project Brain. Connector status lives in a compact strip. Tinted dark neutrals, single teal accent, thin borders.

**Design references:** Linear App, Cursor IDE, Warp Terminal

![Bento grid layout concept — dark mode command center with AI chat as the hero panel, project sidebar, and teal accent highlights](C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/moodboard_main_layout_1776327679143.png)

## 2. Chain of Thought — Transparent Agent Reasoning

Agent reasoning rendered as navigable, collapsible blocks (inspired by Warp Terminal). Each step has a status icon. The active step pulses subtly. Expanded steps reveal tool results in monospace. Collapsed steps show one-line summaries.

**Design references:** Warp Terminal blocks, Cursor Agent mode, Claude thinking indicators

![Chain of Thought panel — collapsible reasoning steps with status icons, active step glow, and expandable JSON detail](C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/moodboard_chain_of_thought_1776327693115.png)

---

## 3. Command Palette — Keyboard-First Navigation

A global `Cmd+K` command palette for instant access to every action, navigation target, and setting. Grouped results, keyboard shortcut badges, backdrop blur.

**Design references:** Linear Cmd+K, VS Code Command Palette, Raycast

![Command palette overlay with grouped actions, keyboard shortcuts, and backdrop blur](C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/moodboard_command_palette_1776327712226.png)

---

## 4. Color & Typography System

The dual-register type system: Inter Variable for human-language content, JetBrains Mono for machine-language content. OKLCH-based palette with tinted neutrals and three candidate accent colors.

**Design references:** Evil Martians OKLCH methodology, GitHub design system, Linear typography

![Design system reference — OKLCH color palette with tinted neutrals and accent variations, Inter and JetBrains Mono type specimens](C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/moodboard_color_typography_1776327746057.png)

## 5. Generative UI — Rich Components in Chat

The AI doesn't just reply with text. It renders interactive components *inside* the conversation: timeline visualizations, pipeline status cards with action buttons, data tables. This turns the chat from a log into a living dashboard.

**Design references:** Vercel v0 Generative UI, Claude Artifacts, ChatGPT Canvas

![Generative UI in chat — embedded timeline visualization and AutoPod pipeline result card with action buttons](C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/moodboard_generative_ui_1776327761479.png)

## 6. Project Brain Sidebar — Structured Knowledge

The persistent project memory rendered as a navigable, searchable knowledge hierarchy. Expandable sections for episodes, characters, style notes, and technical metadata. Active items highlighted with a teal left-border accent.

**Design references:** Notion sidebar, Linear project nav, Obsidian knowledge graph

![Project Brain sidebar — structured knowledge hierarchy with search, expandable sections, and teal accent borders](C:/Users/DSNah/.gemini/antigravity/brain/bb61da8e-3fd5-4836-bef1-da8b3949c943/moodboard_project_brain_1776327774004.png)

## Key Takeaways

| Principle | Visual Expression |
|:---|:---|
| **Information, not decoration** | Every panel serves a purpose; no ornamental elements |
| **Structured transparency** | Chain of Thought is visible but collapsible |
| **Keyboard-first** | Command palette is the primary navigation |
| **Two registers, one voice** | Inter (human) + JetBrains Mono (machine), unified by teal accent |
| **Comfort over spectacle** | Off-white text, tinted neutrals, restrained accent usage |
| **Generative UI** | Rich interactive components embedded in conversation |
